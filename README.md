# Task Manager

A simple Task Manager application built with Python.  
This project demonstrates basic Python concepts such as file handling, OOP, and user input handling.

## Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Save and load tasks from a file

## Requirements
- Python 3.8+

## How to Run
1. Clone this repository:
   ```bash
   git clone https://github.com/Zeinab-Gamal/task-manager.git
   ```

2. Navigate to the project folder:
   ```bash
   cd task-manager
   ```

3. Run the script:
   ```bash
   python task_manager.py
   ```

## Project Structure
```
task-manager/
│
├── task_manager.py   # Main Python script
├── tasks.txt         # Stores tasks (auto-generated after running)
└── README.md         # Project documentation
```
